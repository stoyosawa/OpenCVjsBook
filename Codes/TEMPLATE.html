タイトル部分は H1。体言止め。

他に用法などの説明があれば P を使う。

画像、ビデオ、キャンバスはこの最初の div でくくる。 属性は id - type - width/height - src - class - style - onevent の順。id を tag の横にすると、参照間違いがすくなくなる。 alt は使わない。 タグの id は image、video、camera、キャンバス
input などの制御部分。数値を表示するときにもこれを使う。 input タグの id は xxxInput。xxx 部分は file、size、threshold など、その値の意味。
